# EPS - El Baraka Prestation de Service

Site web professionnel pour EPS, spécialisé dans les services de nettoyage, communication et manutention aéroportuaire à Nouakchott, Mauritanie.

## 🎯 Services Principaux

### 🧹 Nettoyage
- Nettoyage de bureaux et locaux commerciaux
- Nettoyage industriel et entrepôts
- Nettoyage résidentiel
- Nettoyage après travaux
- Désinfection et sanitisation
- Entretien régulier

### 📢 Communication
- Stratégie de communication
- Création de contenu digital
- Gestion des réseaux sociaux
- Publicité et marketing
- Relations presse
- Événementiel

### ✈️ Manutention Aéroportuaire
- Manutention des bagages
- Assistance au sol des avions
- Gestion des cargaisons
- Services de rampe

## 🎨 Palette de Couleurs

- **Bleu principal** : `#003366` (EPS Blue)
- **Blanc** : `#FFFFFF`
- **Rouge accent** : `#FF0000` (EPS Red)
- **Bleu aéroportuaire** : `#1e40af` (Manutention)

## 🛠️ Stack Technique

- **Frontend** : React 18 + TypeScript
- **Styling** : CSS personnalisé (Mobile First)
- **Animations** : Framer Motion
- **Icônes** : Lucide React
- **Build** : Vite
- **Déploiement** : Optimisé pour production

## 📱 Responsive Design

- **Mobile First** approche
- **Breakpoints** : 576px, 768px, 992px, 1200px, 1400px
- **Grille adaptative** : 1 → 2 → 3 colonnes selon l'écran
- **Navigation mobile** avec menu hamburger
- **Animations optimisées** pour chaque device

## 🚀 Installation

```bash
# Cloner le projet
git clone [repository-url]
cd eps-website

# Installer les dépendances
npm install

# Lancer en développement
npm run dev

# Build pour production
npm run build
```
